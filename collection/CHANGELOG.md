# Collection Schema Changelog

All notable changes to the Collection schema will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project uses [Calendar Versioning](https://calver.org/) (YY.MMDD).

## [25.1202] - 2025-12-02

### Added
- Initial collection schema
- Support for longitudinal and multi-task collections
- Renamed `status` to `requirement` for clarity
- Core properties: @context, id, name, title, description, etc.
